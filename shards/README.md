## `dependabot-hex`

Crystal (shards) support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell shards
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd shards && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
